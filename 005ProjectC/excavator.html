Welcome to the world of Excavator and its Friends!

Attention！If you see nothing in the viewports, please check your browser security configuration.

1. Drive the excavator using arrow keys.
2. Change the motion of the excavator’s arm: using keypad “num1/num4” “num2/num5” “num3/num6”.
3. Change camera view using "W/S/A/D" or mouse your mouse. Change FOV using mouse wheel.
4. Change main viewport camera using “1” “2” “3”. Switch between orthographic and perspective camera using “P”.
5. Keeping pressing "1" to switch LookAt target.
6. Show/hide object axis using “Enter”.
Please use a browser that supports "canvas"
